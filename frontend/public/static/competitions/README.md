This directory should contain local copies of competition logos used by the navigation menu.

Place files here (recommended formats: .svg for vector logos, .png for raster):
- premier-league.svg
- la-liga.svg
- bundesliga.svg
- serie-a.svg
- ligue-1.svg
- champions-league.svg
- europa-league.svg

You can add them by downloading the images and placing them in this directory so the app serves them from `/static/competitions/<name>`.
